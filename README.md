### Installation And Running

1. Clone the repo
   ```sh
   git clone https://github.com/speechly/flight-booking-2021.git
   ```
2. Install NPM packages
   ```sh
   npm ci
   ```
3. Run following command in command line`
   ```sh
   npm run dev
   ```
